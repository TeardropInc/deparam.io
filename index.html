deparam.io

Deparameterize URL strings.

deparam.io is a tool to break long URL strings into a readable table of parameters. No longer will you need to visually parse complex URL parameters.

For an example paste the following string into the input:

//wh.consumersource.com/wtd.gif?type=pageview&site=www-02.v-lax.example.com&site_version=www-02.v-lax.example.com_kilo&sess=1390949743367.1390949743367&pickGroup=AR_OW_pick&cg=OW%20LO%20Homepage&ref=&path=%2F&title=Apartments%20for%20Rent%20%26%20Homes%20for%20Rent%20-%example.com%C2%AE%20Apartment%20Finder&fpc=1390949743367&ft=7892.418999996153&cb=0&bs=1312x460&sr=1312x460&os=Mac&browser=Chrome&ver=32&cg_s=ow%20homepage&li=0&set_pick=0d6d5294&get_pick=4b1fe553&shadow_person_id=171705277 ×
